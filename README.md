# RegOS

RegOS is a multi-year enterprise platform. For the hackathon, this is a Minimum Viable Product (MVP) that demonstrates the core innovation: extracting regulatory obligations, turning them into tasks, and ensuring compliance using AI agents.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
